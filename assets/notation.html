Set Notation


Sigma notation is a formal way of describing a set in a concise way. This is very helpful in cases where sets are too large to write out. It uses many of the special sets symbols listed out in the special sets section.

The captial Greek letter Sigma (Î£) is what Sigma notation is named for and is used in the formal declaration.
