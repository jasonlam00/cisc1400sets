Special Sets


By using formal notation to describe sets, we avoid listing all the elements of a set, which would be especially time-consuming and inefficient if the sets were large (or infinite)!


However, before we discuss how to describe sets, let us first introduce certain specific sets that are so common they have been assigned their own descriptive names and abbreviations. You may be familiar with many of these sets already, albeit possibly not aware they were sets.
